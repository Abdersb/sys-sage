\doxysubsection{Detailed Description}
Class \mbox{\hyperlink{class_data_path}{Data\+Path}} represents Data Paths in the topology -- Data Paths represent an arbitrary relation (or data movement) between two Components from the \mbox{\hyperlink{class_component}{Component}} Tree. ~\newline
 Data Paths create a Data-\/\+Path graph, which is a structure orthogonal to the \mbox{\hyperlink{class_component}{Component}} Tree. ~\newline
 Each \mbox{\hyperlink{class_component}{Component}} contains a reference to all Data Paths going to or from this components (as well as parents and children in the \mbox{\hyperlink{class_component}{Component}} Tree). Using these references, it is possible to navigate between the Components and to view the data stores in the Components or the Data Paths.
