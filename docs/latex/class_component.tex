\doxysubsection{Detailed Description}
Generic class \mbox{\hyperlink{class_component}{Component}} -\/ all components inherit from this class, i.\+e. this class defines attributes and methods common to all components. ~\newline
 Therefore, these can be used universally among all components. Usually, a \mbox{\hyperlink{class_component}{Component}} instance would be an instance of one of the child classes, but a generic component (instance of class \mbox{\hyperlink{class_component}{Component}}) is also possible.
